## Description

GraphQL APIs written in Typescript.

## Installation

```bash
$ cp .env.example .env
$ docker-compose up --build
```

## PlayGround

PlayGround will be available on http://localhost:3001/graphql


## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
